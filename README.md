# Kitchen Configurator Pro

Production-ready WordPress plugin for custom kitchen cabinet configuration with WooCommerce integration.

## Requirements

- WordPress 6.0+
- WooCommerce 9.0+
- PHP 8.2+
- MySQL 8.0+
- Composer
